Semantischer Vergleich zweier aussagenlogischer Formeln

Hier können zwei aussagenlogische Formeln auf semantische Äquivalenz überprüft werden. Die Syntax zur Eingabe kannst Du den Beispielen oder der Dokumentation entnehmen.

Erläuterungen

& repräsentiert ∧, dies ermöglicht eine einfache und schnelle Eingabe von Formeln.
Beispiel: A & B = A ∧ B

| repräsentiert ∨.
Beispiel: A | B = A ∨ B

Häufig gestellte Fragen

Welche Syntax wird für die logischen Terme genutzt?

Beispielhaft kann die Syntax den Erläuterungen neben der jeweiligen Funktion entnommen werden. Die detaillierte Beschreibung der Syntax ist in unserer Dokumentation zu finden.

Wie kann ich auf die Schnittstelle zugreifen?

Generell bieten wir keinen uneingeschränkten Zugang zu unserer Schnittstelle, um unsere Server Kapazitäten garantieren zu können. Entwickler können im Entwicklerportal einen Zugangsschlüssel anfordern, zudem bieten wir spezielle Konditionen für Studenten.

Wie kann ich das Projekt unterstützen?

Als Entwickler kannst du über GitHub etwas zum Code beisteuern. Es ist allerdings keine besondere Unterstützung notwendig, die reine Nutzung der Webseite ist Unterstützung genug.

Wofür werden Cookies benötigt/benutzt?

Wir nutzen Google Analytics, um die Nutzung unserer Webseite zu analysieren und um Stolpersteine oder Fehler schnellstmöglich zu beseitigen. Wenn du Google Analytics nicht nutzen möchtest, kannst du dich hier informieren.

Wie finanziert sich dieses Projekt?

Dieses Open Source Projekt finanziert sich durch die Entwickler. Die Webseite ist vollkommen werbefrei.
